I chose a simple web based quiz app


What it does-
This is a trivia quiz app that pulls random questions from a public API. You are given a question with 4 choices to make. If you get the right answer the background changes color and the same thing happens if you choose the wrong answer


What i used for making this app-
* HTML for the skeleton structure
* CSS for the looks and designs
* Js for making everything work
* Public API from opentdb



How to run-

Download this repo and find the folder where you saved it and then double click the file called index.html
After this the quiz should boot up

(make sure you are connected to the internet so that it can fetch the questions from the API)

Alternative way-

If you dont want to go through all the hassle that ive mentioned above then you can simply copy paste this link in any web browser--> https://quizozprog.netlify.app/
The link should work(fingers crossed)



A note-
